Новая версия Chrome включает HTML5 по-умолчанию

Компания Google выпустила стабильную версию десктопного браузера Chrome 56 для операционных систем macOS, Windows и Linux. Это событие произошло спустя 55 дней после выхода Chrome 55.

Главным изменением стало включение технологий HTML5 по умолчанию для всех пользователей. Отметим, что Google начала внедрять HTML5 среди небольшой подборки пользователей еще с релиза Chrome 55, а теперь эта опция включена у всех пользователей в международном масштабе. Поддержка WebGL 2.0 также внедрена по умолчанию.

Статистика использования HTML5 растет с каждым днем

Статистика юзеров HTML поражает! С каждым месяцем цифра пользователей данного языка растет в разы. По сравнения с 2010 годом, она выросла в 5 раз. Средний возраст юзеров, по статистике - 21 год. Скоро HTML5 захватит всю видимую часть Интернета.

Билл Гейтс назвал HTML5 стандартом будущего Интернета

Билл Гейтс на прошедшей выставке GDC2017 сделал горячее заявление: "HTML5, я уверен, станет всеобъемлющим стандартом будущих интернет-технологий мира!"

Помимо этого он показал собственные разработки в этой области, открыв четыре новых тэга и применил их прямо в риал-тайм разработке сайта. Впечатляет, не правда ли?
